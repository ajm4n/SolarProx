What is SolarProx?

Welcome to Force Of Mercury's latest project, SolarProx. SolarProx is a project designed to use an assortment of libraries to allow Proxmox, a virtual environment for running managing virtual machines, to take advantage of active directory for the usage of managing a lab for penetration testing.
